# 🎴 Бура — Telegram Bot

Карточная игра **Бура** для 2–4 игроков. Написана на **aiogram 3**.  
Принцип: у каждого игрока ровно **одно** сообщение-экран. Все обновления — `edit_message_text`. Никакого мусора в чате.

---

## Быстрый старт

```bash
pip install aiogram==3.13.1
```

Открой `bot.py`, вставь токен:
```python
BOT_TOKEN = "123456:ABC-токен-от-BotFather"
```

Запуск:
```bash
python bot.py
```

---

## Команды

| Команда | Описание |
|---------|----------|
| `/new` | Создать новую сессию (хост) |
| `/join XXXXX` | Войти по 5-значному коду |
| `/leave` | Покинуть текущую сессию |
| `/start` `/help` | Правила и команды |

---

## Сценарий игры

```
Хост:          /new
               → видит экран лобби с кодом, напр. "R7K2A"
               → нажимает кнопку числа игроков (2 / 3 / 4)

Игрок 2:       /join R7K2A
               → видит лобби

(Игрок 3, 4:   /join R7K2A  — опционально)

Хост:          нажимает ▶️ Начать игру

--- Игра началась ---

Чей ход:       видит свои карты кнопками → нажимает карту (выделяется ✅)
               → нажимает ▶️ Сыграть
               → ВСЕ экраны обновляются (edit_message)

Не чей ход:    видит карты в тексте + кнопку 🔄 Обновить

Конец раунда:  Хост нажимает 🔄 Новый раунд

Победитель:    первый набравший 31 очко
               → кнопка 🏠 В главное меню
```

---

## Принцип «чистого чата»

```
/new           → удаляется сразу (safe_delete)
               → bot.send_message()  ← ЕДИНСТВЕННОЕ сообщение игрока

Каждый ход:    bot.edit_message_text()  ← тот же message_id

/join          → удаляется сразу
               → bot.send_message() для нового игрока
               → edit_message_text() для остальных

Ошибки:        cb.answer(show_alert=True)  ← всплывающий popup, не сообщение

Конец игры:    edit_message_text()  ← финальный экран без кнопок
```

---

## Правила Буры

- Колода 36 карт (6 7 8 9 10 J Q K A × 4 масти)
- Раздаётся по 3 карты, после каждой взятки — добор до 3
- Козырная масть бьёт любую другую
- **Обязан ходить в масть** вышедшей карты, если есть
- Если своей масти нет — обязан **козырять**
- Очки: A=11, 10=10, K=4, Q=3, J=2, остальные=0
- **Побеждает первый**, кто набирает **31 очко** суммарно по раундам
